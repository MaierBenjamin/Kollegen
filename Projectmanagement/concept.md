# Konzept Kollegen (Teams Klon)

## Idee
Kollegen soll eine alternative zu Programmen wie Teams, 

## Funktionalität
- Konto erstellen
- Register
- Login/Logout
- Kontoeinstellungen und Nutzerdaten verwalten
- Chatfunktion
- Rollen/Berechtigungssystem
- Organisationsbeitrittsverwaltung via Nutzerkonto
- Mannschaften (Übergeordnet Kanäle)
- Kanäle (in Mannschaften)
- Dateisharing (in Mannschaften)

## Technologien
| Bereich               | Technologie / Tool      |
|-----------------------|--------------------------|
| Frontend              | Vue                      |
| Backend               | Node                     |
| Datenbank             | MariaDB                  |
| Versionskontrolle     | GitHub                   |
| Projektmanagement     | Kanban (MIRO)            |
| DevOps/Pipelines      | GitHub Actions           |
| Server                | Selfhosting (Ubuntu)     |
